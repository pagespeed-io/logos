# pagespeed.io Logos

## Usage

See `/svg/ and `/png/` folders for different formats. Prefer `.svg` over `.png`, if possible.


## Source

The `logos.sketch` file contains the source from which custom dimensions can be exported.


## Background Colors

```
1. #ffffff       White
2. #2a2c39       Dark Blue
3. #9ba1b3       Dark Grey
4. #dde1e9       Light Grey
```
